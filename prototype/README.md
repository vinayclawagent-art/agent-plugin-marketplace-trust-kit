# Agent Plugin Marketplace Trust Kit Prototype

Source: [[Grok Build Plugin Marketplace as Agent App Store]]  
Package: [[Agent Plugin Marketplace Trust Kit]]

## What this demonstrates
A small, fillable artifact that turns the X signal into operator evidence instead of a passive note.

## How to use
1. Open [[Agent Plugin Marketplace Trust Kit/Plugin Trust Review Packet]].
2. Fill the scenario, inputs, constraints, evidence, and decision fields.
3. Attach manifest/repo/permission/sandbox evidence before claiming the plugin is safe.
4. Fill [[Agent Plugin Marketplace Trust Kit/Plugin Trust Promotion Decision Card]] before choosing install / sandbox / iterate / reject.
