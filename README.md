# Agent Plugin Marketplace Trust Kit

Agent plugin marketplaces are becoming app stores for skills, MCPs, hooks, and commands; the useful artifact is a trust-and-install review kit before running third-party code.

Inspired by: https://x.com/xai/status/2065099299541893577

## What it contains
- `artifact-package/README.md` — Obsidian package mirror.
- `prototype/` — fillable first-trial packet.
- `infographic/` — workflow explanation.
- `skill-draft/` — unpromoted Hermes skill draft.
- `improvement-loop/` — recurring improvement loop mirror.

## Status
Template-ready, not validated. Use the prototype on a real task before promoting the workflow.

## Next focus
Use the trust review packet on one real marketplace plugin, attach manifest/repo/permission evidence, then decide install / sandbox / reject.
