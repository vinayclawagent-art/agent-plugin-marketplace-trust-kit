# Agent Plugin Marketplace Trust Kit Prototype

Source: [[Grok Build Plugin Marketplace as Agent App Store]]  
Package: [[Agent Plugin Marketplace Trust Kit]]

## What this demonstrates
A small, fillable artifact that turns the X signal into operator evidence instead of a passive note.

## How to use
1. Start with [[Agent Plugin Marketplace Trust Kit/Trial Operator Handoff Checklist]] so the trial runner follows the existing packet → evidence → decision → debrief sequence.
2. Open [[Agent Plugin Marketplace Trust Kit/Plugin Trust Review Packet]].
3. Fill the scenario, inputs, constraints, evidence, and decision fields.
4. Grade the captured proof with [[Agent Plugin Marketplace Trust Kit/Plugin Trust Evidence Quality Rubric]] before changing trust claims.
5. Attach manifest/repo/permission/sandbox evidence before claiming the plugin is safe.
6. Fill [[Agent Plugin Marketplace Trust Kit/Plugin Trust Promotion Decision Card]] before choosing install / sandbox / iterate / reject.
